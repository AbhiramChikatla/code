7.arp
Description: arp is a command-line utility used to view and manipulate the ARP (Address Resolution
Protocol) cache. ARP is a protocol that translates IP addresses into MAC addresses (the physical hardware
addresses used by network devices to communicate on a local network). This translation is necessary because
while IP addresses are used to route packets between networks, MAC addresses are used to deliver packets
within the same local network segment.
